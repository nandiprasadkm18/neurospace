**NeuroScape** is a futuristic, high-fidelity productivity engine that transforms goal-setting into a 3D spatial experience. Designed for builders, thinkers, and deep-focus operatives, it merges cinematic UI with context-aware AI to map your mental universe.

---

## 🚀 CORE PROTOCOLS

### 🪐 1. THE UNIVERSE (3D Goal Mapping)
- **Spatial Computing**: Visualize your life goals as planets in a 3D solar system.
- **Task Satellites**: Individual tasks orbit their parent protocols, providing a gravitational sense of priority.
- **Neural Paths**: Constellation mode connects related objectives to visualize focus clusters.

### ✍️ 2. THE DREAM WORKSPACE (Deep Work)
- **Flow Orchestration**: A distraction-free environment with a persistent global timer.
- **Neural Logging**: Automatic session timestamping and emotion-based note crystallization.
- **Adaptive UI**: The interface shifts based on your current focus score and activity intensity.

### 🏛️ 3. THE MEMORY VAULT (Long-term Archival)
- **3D Time-Capsule**: Navigate your past thoughts in a floating 3D timeline.
- **Crystallized Data**: Every note is stored with rich metadata (date, duration, emotion).
- **Infinite Scroll**: A searchable history of your mental evolution.

### 🧠 4. ARIA CORE (Integrated Intelligence)
- **Contextual Intelligence**: An AI assistant with direct access to your active tasks and focus history.
- **Strategic Direction**: Real-time recommendations on what task to prioritize based on your "Neural Momentum."
- **Sentiment Reactivity**: The platform's visual mood shifts as ARIA analyzes the emotional tone of your notes.

---

## 🛠️ TECH STACK

- **Frontend**: React 18, Vite, Three.js, React Three Fiber, Framer Motion.
- **State Management**: Zustand (with full persistence).
- **Backend**: Node.js, Express, Groq SDK (Llama 3).
- **Database**: MongoDB.
- **Styling**: Vanilla CSS with "Deep Space Luxury" glassmorphism.

## ⚡ QUICK START

### 1. Clone the Neural Link
```bash
git clone https://github.com/nandiprasadkm18/neurospace.git
```

### 2. Configure the Core
Create a `.env` file in the root directory:
```env
GROQ_API_KEY=your_key_here
MONGODB_URI=your_mongodb_uri
```

### 3. Initialize Protocols
```bash
# Install dependencies
npm install

# Start the full system (Frontend + AI Backend)
npm run dev:all
```
